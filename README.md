This study is about solving a real problem that many businesses face people on the inside misusing their access to steal or manipulate records. We built two systems to address this: StockSecure POS which makes sure each employee can only access the parts of the system that their job actually requires so a cashier can't secretly edit inventory or erase a sale and CyberLedger which keeps a secure log of every transaction so that nothing can be quietly changed or deleted without leaving a trace. The goal of our study is to show that by combining strict access control with a reliable audit trail businesses can significantly reduce internal fraud and keep their records accurate and trustworthy.
